# Repo Role

This repository is the public-facing SNAP codebase intended for GitHub and
other release-facing distribution.

## Purpose

- Publish product code, stable docs, examples, adapters, tests, and release-safe assets
- Accept reviewed code and documentation ported from the private SNAP workspace

## Do Not Add

- `PHASE*.md` prompt/result workflow files
- `0. development/`, `1. launch/`, or action-item archives
- `devnet-results/`, `adversarial-results/`, or `stress-results/`
- personal notes, launch checklists, budget drafts, credentials, or key material
- seed note journals, relayer databases, local ledgers, or machine-specific output
- absolute local filesystem paths

## Working Rule

Feature development and internal planning happen in the separate private SNAP
workspace. Only reviewed public-safe code and docs should be ported here.
